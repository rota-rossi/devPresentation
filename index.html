What are the basics?

  • Primitive types (numbers, strings, booleans, arrays)
  • Composite types (structs, objects, enums, several names...)
  • Repetitions (for, while,...)
  • Conditionals (if, case, ...)
  • Code blocking (functions, methods, ...)
